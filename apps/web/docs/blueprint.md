# **App Name**: First Impression

## Core Features:

- Text Display: Display 'Hello World' prominently on the screen.

## Style Guidelines:

- Background color: Soft, desaturated blue (#D1E0E9) to create a calm and inviting backdrop. This muted tone will help the text stand out without being too harsh.
- Primary color: Medium-dark blue (#36607A) to provide strong contrast against the light background. It evokes feelings of stability and clarity.
- Accent color: Teal (#48A9A6) to introduce a touch of vibrancy and modernity, drawing attention to key elements without overwhelming the overall design.
- Font: 'Inter', a sans-serif font, for a clean and modern look that ensures excellent readability, suitable for both headlines and body text.
- Simple, centered layout with plenty of whitespace to focus on the essential text.
- A subtle fade-in animation for the text on initial load.
